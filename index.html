Abstract argumentation considers an argument framework to be a directed graph where nodes represent arguments and edges represent conflicts between the arguments. Several algorithms can be used to generate sets of coherent algorithms from an argument framework, known as extensions.

Demo 1 - Review

Review the available extensions in the distinct graphs with up to three arguments.

Demo 2 - Test

Test and share your own argument frameworks.

Demo 3 - Random

Review the available extensions in randomly generated argument frameworks.

These demos use the excellent Visjs to visualise argument frameworks as networks.
